Ivory&Co — Responsive Store Landing Page
A responsive lifestyle store landing page built with HTML, CSS, and Bootstrap 5.

Project Overview
Ivory&Co is a front-end web project for a fictional lifestyle store. The page is designed with a warm, pastel brown and cream aesthetic and is fully responsive across desktop and mobile screen sizes.

Pages & Sections
SectionDescriptionNavbarSticky top navigation with links to each section. Collapses into a hamburger menu on mobile.HeroFull-width intro section with a headline, description, and Shop Now button.ProductsA grid of 4 product cards, each with an image, category, name, description, price, and Shop Now button.NewsletterEmail sign-up section with an input field and Subscribe button.FooterBrand name, tagline, copyright, and legal links.

Design Choices

Warm neutral palette — beige, cream, and brown tones create a calm, lifestyle feel.
Playfair Display font — an elegant serif that gives the brand an editorial, boutique look.
Square corners (border-radius: 0) — used on buttons and cards for a clean, structured aesthetic.
Hover effects — cards lift slightly and buttons change color on hover to improve interactivity.
Bootstrap grid — col-6 col-lg-3 makes the product cards show 2 per row on mobile and 4 per row on desktop.


How to Run

Download or clone the project folder.
Make sure all image files are inside an image/ folder in the same directory.
Open index.html in any web browser — no installation needed.
